* SRFI 179: Nonempty Intervals and Generalized Arrays

** by Bradley J. Lucier

This repository hosts a forked and modified version of [[http://srfi.schemers.org/srfi-122/][SRFI 122]]: Nonempty Intervals and Generalized Arrays, a [[http://srfi.schemers.org/][Scheme Request for Implementation]].

This SRFI is in /final/ status.

The full documentation for this SRFI can be found in the [[http://srfi.schemers.org/srfi-179/srfi-179.html][SRFI Document]].

If you'd like to participate in the discussion of this SRFI, or report issues with it, please [[http://srfi.schemers.org/srfi-179/][join the SRFI-179 mailing list]] and send your message there.

Thank you.
